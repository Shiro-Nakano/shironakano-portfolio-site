## 本仕様書に関する注意事項
- 本仕様書は要求仕様と画面・技術仕様を兼ねる

## 1. 機能要件（Functional Requirements）

- ユーザーが仲野朱郎の自己紹介・制作物・依頼時の概要を理解できる内容である
- ユーザーからの問い合わせをサイト経由で受け取り可能
- ファン・興味層と依頼検討層の両方とものユーザーが満足できる内容である


### 1-1. ページ・セクション構成　

| ID | セクション名 | 構成概要 |
|----|-------------|------|
| F-01 | Hero（トップ） | メインのイラスト・CATボタン |
| F-02 | Works（制作物一覧） | 制作物・各詳細 |
| F-03 | Profile（自己紹介） | 自己紹介文・スキルセット・経歴 |
| F-04 | Service(依頼時概要) |　依頼内容に応じた工程・期間等の概要 |
| F-05 | Contact | 問い合わせフォーム・各SNSリンク |

### 1-2. 機能一覧

| ID | 機能名 | 説明 | 優先度 |
|----|--------|------|--------|
| F-10 | ナビゲーション | 各セクションへのスムーズスクロール | 必須 |
| F-11 | ハンバーガーメニュー | トグル表示する | 必須 |
| F-12 | ダークモード切替 | ボタンでライト／ダーク切替 | 必須 |
| F-13 | Works モーダル | 対象をクリックで制作物の詳細を表示する | 中 |
| F-14 | Worksフィルター | タグ（HTML/CSS/TS等）で制作物を絞り込む | 低 |
| F-15 | トップへ戻るボタン | スクロール量に応じて表示・非表示を制御する | 低 |

### 1-3. コンテンツ要件

| ID | コンテンツ | 形式 | 備考 |
|----|-----------|------|------|
| C-01 | キャッチコピー | テキスト | Hero・ページタイトルに使用 |
| C-02 | 自己紹介文 | テキスト（200字程度） | Profile セクション |
| C-03 | オリジナルイラスト | 画像 | Hero セクション・Works セクション |
| C-04 | スキルリスト | テキスト＋アイコン | HTML / CSS / TypeScript / Figma 等 | Profile セクション・Works セクション |
| C-05 | 制作物データ | タイトル・説明・使用技術・URL・サムネイル | Works セクション |
| C-06 | 依頼概要 | 工程・期間 |テキスト・イラスト| Service セクション |
| C-07 | 連絡先・SNSリンク | 問い合わせフォーム・URL（Instagram / X  等） | Contact セクション |

## 2. 非機能要件（Non-Functional Requirements）

### 2-1. パフォーマンス

| ID | 項目 | 基準値 | 計測ツール |
|----|------|--------|-----------|
| NF-01 | ページ読み込み時間 | 3秒以内（4G回線想定） | PageSpeed Insights |
| NF-02 | 画像最適化 | WebP形式・適切なサイズ指定 | 目視確認 |

### 2-2. アクセシビリティ

| ID | 項目 | 基準 |
|----|------|------|
| NF-10 | カラーコントラスト比 | WCAG AA準拠（4.5:1以上） |
| NF-11 | キーボード操作 | Tab キーで全操作が可能 |
| NF-12 | セマンティックHTML | 適切な見出し階層・ランドマーク |
| NF-13 | 画像の代替テキスト | 全画像に alt 属性を付与 |

### 2-3. 対応環境（ブラウザ・デバイス）

| 分類 | 対象 |
|------|------|
| ブラウザ | Chrome（最新）/ Safari（最新）/ Edge（最新） |
| デバイス | PC / タブレット / スマートフォン |
| 画面幅 | 320px 〜 1440px |

### 2-4. SEO・メタ情報

| ID | 項目 | 内容 |
|----|------|------|
| NF-20 | ページタイトル | `仲野朱郎 | Portfolio` 形式 |
| NF-21 | meta description | 120字以内の自己紹介文 |
| NF-22 | OGP設定 | SNSシェア時のサムネイル・タイトル表示 |
| NF-23 | favicon | SVGまたはPNG（32×32px） |

### 2-5. セキュリティ

| ID | 項目 | 対応内容 |
|----|------|---------|
| NF-30 | 外部リンク | `target="_blank"` に `rel="noopener noreferrer"` を付与 |
| NF-31 | 個人情報 | メールアドレスは直接記載せずSNS・フォーム経由とする |

## 3. 技術仕様

### 3-1. 使用技術

| 分類 | 技術 | バージョン |
|------|------|-----------|
| マークアップ | HTML5 | - |
| スタイル | CSS3 | - |
| スクリプト | TypeScript | 6.0 |
| ビルドツール | Vite（Vanilla） | 6.x |
| パッケージ管理 | npm | - |
| バージョン管理 | GitHub | - |
| ホスティング | LoliPOP！レンタルサーバー | - |

| イラスト制作 | CLIP STUDIO PAINT EX | Ver5.0.2 |

### 3-2. 制約・前提条件

| 項目 | 内容 |
|------|------|
| フレームワーク | React / Vue 等のフレームワークは使用しない |
| 外部ライブラリ | 原則使用しない（アイコンのみ例外：Font Awesome 等） |
| 予算 | 無料ツール・サービスのみ使用 |
| 対応言語 | 日本語のみ（多言語対応はスコープ外） |
| 開発期間 | 2025-04-06 〜 2025-04-27（約3週間） |
