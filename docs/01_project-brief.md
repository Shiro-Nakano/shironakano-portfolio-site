# プロジェクト概要書（Project Brief）

**プロジェクト名：** 仲野朱郎のポートフォリオサイト  
**作成日：** 2025年4月14日
**作成者：** 仲野　朱郎
**バージョン：** 1.0

---

## 1. プロジェクトの背景・目的


研修で習得したHTML / CSS / TypeScriptのスキルと自身の創作物を対外的に示すため、
自身の制作物・経歴・スキルセットをまとめたポートフォリオサイトを制作する。

---

## 2. ターゲットユーザー


| 優先度 | ターゲット | 想定シーン |
|--------|-----------|-----------|
| 高 | 営業担当・役員の方 | 面談時に制作物を評価する |
| 高 | クライアント様 | 作品と技量を評価する |
| 中 | 他サイト閲覧者 | 作品や経歴を知ってもらう |

---

## 3. ゴール定義

### 3-1. ビジネスゴール

- IT移行可能な実務レベルに遜色ない作品を作り上げる
- 過去・現在・未来の自身の技術スタック・制作物を一箇所にまとめるポートフォリオサイトに仕上げる
- 継続的に更新できる仕組みを持つ

### 3-2. ユーザーゴール

- 制作者のスキルレベルと得意分野を3分以内に把握できる
- 過去の制作物の概要・使用技術・成果物を確認できる
- 依頼後にどのような流れで制作するのかのイメージができる
- 連絡先・SNSリンクにすぐアクセスできる

### 3-3. 成功指標（KPI）

| 指標 | 目標値 | 計測方法 |
|------|--------|---------|
| 公開期日 | 2026-04-27 | 目視確認 |
| ページ読み込み時間 | 3秒以内 | PageSpeed Insights |

---

## 4. スコープ（対象範囲）

### 含む（In Scope）

- トップセクション
- 自己紹介セクション
- 制作物一覧セクション
- 依頼時の工程概要セクション
- お問い合わせ / SNSリンクセクション
- レスポンシブ対応（PC / タブレット / スマートフォン）
- ダークモード対応

### 含まない（Out of Scope）

- ブログ機能
- CMS連携
- バックエンド・データベース
- 多言語対応（日英切り替えなど）
- アニメーション・3D演出の過度な実装

---

## 5. 技術スタック

| 分類 | 採用技術 | 理由 |
|------|---------|------|
| マークアップ | HTML5 | セマンティックな構造化 |
| スタイル | CSS3（カスタムプロパティ・Flexbox・Grid） | 標準技術の習熟 |
| スクリプト | TypeScript | 型安全・実務標準 |
| ビルドツール | Vite | 高速・設定が軽量 |
| Viteフレームワーク | Vanilla | 制作が容易 |
| バージョン管理 | Git / GitHub | 変更履歴の管理 |
| ホスティング | LoliPOP！レンタルサーバー  | クライアント指定の提出用サーバー |
| デザインツール | Figma | ワイヤーフレーム・UI設計 |
| イラスト制作ツール | ClipStudio | イラスト制作 |

---

## 6. 制約・前提条件

- **スキル制約：** HTML / CSS / TypeScript を使用し、フレームワーク（React等）は使用しない
- **予算：** 無料ツール・サービスのみ使用
- **期間：** 2025-04-06 〜 2025-04-27（約3週間）
- **デザイン方針：** 近代的なデザインの上、ユーザビリティーを意識したもの

---

## 7. リスクと対策

| リスク | 影響度 | 対策 |
|--------|--------|------|
| イラスト制作に時間をかけすぎる | 高 | 仮イラストを先に仮配置し、制作を進める |
| デザインに時間をかけすぎる | 高 | ワイヤーフレームを先に確定してからデザインに入る |
| ブラウザ互換性の問題 | 低 | Chrome / Safari で定期的に動作確認する |
